# Dashboard Streamlit (Déprécié)

Ce dossier contenait initialement le dashboard Streamlit de l'application AirSentinel.

Conformément à la nouvelle structure du projet, le dashboard a été supprimé pour privilégier les deux composants principaux :
1. **API** : Backend FastAPI.
2. **PWA** : Application web progressive (Frontend).

Le README est conservé comme point de repère.
