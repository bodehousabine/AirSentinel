- Nécessite un 2ème service Render ou un service séparé

---

## Étapes pour déployer le Backend sur Render

### 1. Prérequis
- Compte Render.com
- Repo GitHub pushes

### 2. Sur Render.com
1. Aller sur [Render Dashboard](https://dashboard.render.com)
2. Cliquer "New" → "Blueprint"
3. Connecter le repo GitHub
4. Sélectionner le fichier `render.yaml`
5. Cliquer "Apply"

### 3. Configuration requise
- **DATABASE_URL** : Configurer manuellement dans le dashboard Render
  - Aller dans "Environment" du service
  - Ajouter la variable `DATABASE_URL` avec l'URL Supabase

### 4. Variables d'environnement recommandées
```
DATABASE_URL=postgresql://user:pass@host:5432/dbname
DEBUG=false
PORT=8000
```
